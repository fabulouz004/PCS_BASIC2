MEANING OF CLASS AND CLASS MONITORS


A class is a group of students or pupils who are taught together. A class monitor is a person whom a teacher trusts unless he/she was chosen by student vote And they have to maintain class in silence when the teacher is not around. All the class Monitors have the right and duties to assist the class and subject teachers to maintain a good order in their own classroom.

DUTIES OF A CLASS MONITOR

1. Make sure the class is in order.
2. Sit in front seats.
3. Make sure the class is neat.
4. Clean the board.
5. Write names of noise makers when the teacher is not in class.
6. Clean the teachers seats.

QUALITIES OF A CLASS MONITOR

1. Punctuality
2. Regularity
3. Obedience
4. Neat
5. Intelligent
6. Hardworking
7. Polite.
